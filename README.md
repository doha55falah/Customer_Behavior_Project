One-line summary:
End-to-end data analytics pipeline: load dataset in Python, perform EDA & cleaning, push cleaned data to MySQL, run analytical SQL queries, build a Power BI dashboard.

Overview

This project demonstrates a complete analytics workflow recruiters love: data ingestion, exploration, cleaning, persistence to a relational DB (MySQL), query-driven analysis, visualization (Power BI). It’s reproducible, documented, and suitable for portfolio presentation.

Tools & Tech Stack

.Python 3.8+ (pandas, numpy, matplotlib / seaborn, Jupyter/Colab)

.MySQL  — storage + analytical queries

.SQLAlchemy + PyMySQL (Python ↔ MySQL integration)

.Power BI Desktop (connect to MySQL to build dashboard)
